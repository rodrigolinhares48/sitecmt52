# CMT LII – UEF Alice Mendes

Site institucional do Colégio Militar Municipal Tiradentes LII – UEF Alice Mendes, desenvolvido em HTML5, CSS3 e JavaScript puro.

## Páginas incluídas

- Início
- Histórico
- Filosofia, missão e valores
- Nossa equipe
- Notícias e páginas individuais de matérias
- Atividades ofertadas
- Calendário letivo interativo
- Horários de aula
- Documentos
- Contato
- Página 404

## Publicação

O projeto pode ser publicado em GitHub Pages, Cloudflare Pages, Netlify, Hostinger ou qualquer hospedagem de arquivos estáticos.

## Antes de publicar

1. Inserir telefone, e-mail, endereço completo e horários oficiais em `contato.html`.
2. Conferir nomes, postos, funções e links da equipe em `equipe.html`.
3. Atualizar documentos na pasta `assets/documentos` quando houver novas versões.
4. Conectar o formulário de contato a um backend ou serviço de formulários, caso seja necessário receber mensagens.

## Estrutura

- `assets/css/style.css`: estilos e responsividade.
- `assets/js/main.js`: menu, filtros, animações, calendário, compartilhamento e validação.
- `assets/images`: imagens otimizadas em WebP e ícones.
- `assets/documentos`: calendário e horários em PDF.
